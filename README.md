# WordMasterProject1
실프 프로젝트1

영어 단어장 프로젝트를 제작하였다.

1. 모든 단어보기
2. 수준별 단어 보기
3. 단어 검색
4. 단어 추가
5. 단어 수정
6. 단어 삭제
7. 파일 저장
8. 나가기

1~8까지의 총 8개의 메뉴가 있으며 아래는 각각의 번호에 따른 실행 화면이다.
